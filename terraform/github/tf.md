## Github Provider

- Create `github` module following [the tutorial](https://dev.to/pwd9000/manage-and-maintain-github-with-terraform-2k86).

- Run

  ```bash
  terrform init
  terraform validate
  terraform fmt
  ```

- Import labs repo

Enter ```terraform import github_repository.labs Devops-labs```
Resutls:
